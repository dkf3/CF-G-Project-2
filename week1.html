Week One

HyperText Markup Language

In our first class, we started by briefly discussing what the Internet is. We talked about different types of websites, both static and dynamic and the differences between the two.

We then moved on to learning about HTML, a coding language which creates and describes the basic structure and content on a web page.

In HTML, you use brackets and tags to tell the computer how to perform certain functions. Tags have both an opening and closing bracket, with the content between the two.

For example, the command < p1 > creates a new paragraph of text, as shown below, with both a opening and closing tag, and the message you would like to write in the middle.

HTML Basics

All HTML files must start with the tag < !DOCTYPE html > to let the computer know which language you are using in your file.

Next, place a < html > tag in which all of your content will be enclosed. But from that point you can start adding content in your HTML file.

HTML documents have a head and a body, each categorized by a < head > and a < body > tag respectively. Within the head of an HTML doc, place titles and formatting tags to help structure your website.

The main content of the website, what you want viewers to see, should be placed in the body. This can include things such as images, text, headings, and links.
