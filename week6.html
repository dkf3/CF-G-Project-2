Week Six

Javascript and JQuery

Javascript:

Javascript is one of most important and useful programming languages.

Javascript allows the programmer to make functions which prompt the website user to provide information, which can then be used by the developer in any number of ways.

This helps to make websites more interactive and user friendly. Further, this allows the developer to make a dynamic, instead of static, website.

JQuery:

However; while useful, Javascript is a very complex, verbose programming language with many functions and abilities.

JQuery is pre-compiled library of Javascript which can be downloaded online.

JQuery was made to be efficient and usable, providing all of the benefits of Javascript, but without the difficulty of finding, creating, or manipulating specific functions.
