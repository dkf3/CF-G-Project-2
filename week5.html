Week Five

Bootstrap

In Week 5, students were introduced to Bootstrap, an online framework which allows for easier stylizing of websites.

Bootstrap vs. CSS:

Both Bootstrap and CSS were made to stylize other types of code. However, Bootstrap was set up to be more user friendly than CSS.

CSS is a very useful resource, however it is quite finnicky and sometimes requires a lot of code to do relatively simple things. Additionally, CSS has a tendency to look very different depending on the size of the screen you are using to view the website.

Bootstrap is an online, downloadable library of code which contains many functions and shortcuts, allowing for stylizing to be a lot more efficient than CSS.

More About Bootstrap:

Within Bootstrap, it is very simple to create grids, columns, charts, buttons, and nav bars, which was relatively complicated to create using CSS.

Further, Bootstrap is easy to override. If given a Bootstrap template, you can easily change many elements of the style while keeping others. This makes creating professional looking websites a lot easier than with CSS!
