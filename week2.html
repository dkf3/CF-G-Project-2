Week Two

Cascading Style Sheets (CSS)

CSS:

CSS is a programming language which allows a developer to stylize their HTML code.

In Week 2, we were introduced to CSS. The first thing we learned was how to link your CSS file into your HTML document so that you can use the two simultaneously.

Using CSS, you can add background colors and images to a page. To do so, you need to use a selctor and target the element you want to style.

In the image below, the h1 tag is being selected, changing the background color of the text box to yellow and changing the text color to blue.

Image From MDN Web Docs

You can also change the color, alignment, size, and font of text on your webpage using CSS, as well as create and stylize many other elements.
