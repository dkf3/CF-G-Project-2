Week Three

User Experience (UX)

In Week 2, we were introduced to User Experience (UX). UX was defined as the overall feelings your product, website, tool, or application inflicts on those who are using it.

The Importance of UX:

The UX of an app or a website can make or break a product. No matter how important the content of a website is, if it is not accessible, users will not be able to properly use the website.

Here are some ways to make a website more user friendly:

  • Add a home page which can easily be found
  • Create a navigation bar where you can see all pages of the website
  • Make sure the website is compatible with multiple formats, ie. phones or different sized screens
